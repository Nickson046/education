📗 How to Import into Google Sheets

  1. Click Download CSV above
  2. Open Google Sheets → Create new spreadsheet
  3. Click File → Import → Upload → select the CSV file
  4. Choose "Replace spreadsheet" → Import data
  5. Your data is live in Sheets — sortable, filterable, shareable
